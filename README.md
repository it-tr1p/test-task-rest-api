# test-task-rest-api

```
Склонировать
git clone git@github.com:it-tr1p/test-task-rest-api.git
cd test-task-rest-api

Установить зависимости
poetry install

Активировать виртуальное окружение
poetry shell

изменить название .env.template >> .env
mv .env.template .env
написать настройки своей бд

создать таблицы запустив скрипт
/app/db/create_db.py
запустите main.py
```
