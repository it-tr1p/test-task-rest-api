# test-task-rest-api

```
Склонируйте
git clone git@github.com:it-tr1p/test-task-rest-api.git
cd test-task-rest-api 
Установите зависимости
poetry install
Активировать виртуальное окружение
poetry shell
измените название .env.template >> .env
mv .env.template .env
напишите настройки своей бд
создайте таблицы запустив скрипт
/app/db/create_db.py
запустите main.py
```
